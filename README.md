# 防災調整池 計算ソフト（claude-pondsoft）

茨城県・千葉県の基準に対応した防災調整池（調整池・調節池）の容量計算Webアプリです。
ブラウザで条件を入力して計算し、**PDF計算書**（設計条件／流域／貯留施設／洪水吐き／総括表）を出力します。

## 機能

- **降雨強度式**: 県・地区・確率年を選ぶと自動設定（r = a/(tⁿ+b) 形式）
  - 茨城県: 館野・水戸の2地区（出典: 茨城県 道路計画設計マニュアル 表8-3-7・8-3-8）
  - 千葉県: 我孫子・横利根・銚子・松戸・千葉・勝浦・館山の7地区
    （出典: 千葉県における宅地開発等に伴う雨水排水・貯留浸透計画策定の手引 巻末資料－1）
- **計画降雨波形**: 中央集中型／後方集中型（継続時間24時間標準）
- **流入ハイドログラフ**: 合理式連続モデル（洪水到達時間は直接指定または土研式）
- **浸透施設**（任意）: 有効降雨モデル。設計浸透量 R（施設種別×数量×単位設計浸透量の積み上げ、
  または R・設計浸透強度 Fc の直接入力）から Fc=R/(Ac×10) を算定し、有効降雨強度 f·I から
  差し引いて流入を低減。必要調節容量の縮小効果を反映（千葉県手引 第10条準拠）
- **貯留追跡計算（厳密法）**: H-A-V（せつ頭錐体／平均面積柱体）＋オリフィス（矩形・円形、複数段）
  による Δt=1分 の追跡計算。最大放流量・必要調節容量・H.W.L を算定
- **簡便法**: 大規模宅地開発に伴う調整池技術基準（案）の一般式 V=(rᵢ−r꜀/2)・f・A・tᵢ/6
- **洪水吐き**: 設計洪水流量（1/200等 × 1.2倍）、越流幅～越流水深表、余裕高チェック
  （千葉県: 掘り込み式0.3m／ダム式は技術基準(案)相当、地震波高 he=kτ/π・√(gH)）
- **堆積土砂**: 150 m³/ha・年 × 造成期間を総容量に加算
- **入力チェック**: 計算前に H-A-V の単調性・面積・オリフィス敷高範囲・越流頂＜天端などを
  検証し、不正時は日本語メッセージで通知（送信前のクライアント側即時チェック＋サーバ側で最終防衛）
- **プロジェクト保存/読込**: 入力一式を JSON で保存・復元（設計変更の試行錯誤に。外部依存なし）
- **結果グラフ**: 計算結果画面に流入・放流ハイドログラフ＋水位を重ね描き（ネイティブCanvas）
- **PDF計算書**: IPAゴシック埋め込み。ハイドログラフ図・池の縦断模式図・時系列表付き

## 起動方法

```bash
pip install -r requirements.txt
uvicorn app.main:app --reload
# → http://127.0.0.1:8000 をブラウザで開く
```

## テスト

```bash
pytest
```

降雨強度式は基準書の数値表（例: 館野1/50・t=60分 → 84.74 mm/hr、千葉地区1/50・t=60分 → 72.9 mm/hr）
と照合しています。貯留追跡は水収支・例示値（千葉県手引 図5-2）との整合を確認しています。

## 県プリセットの編集・追加

`data/prefectures/*.json` に県ごとの降雨強度式と基準ルールを定義しています。
係数の修正や県の追加はJSONの編集のみで行えます（アプリ再起動で反映）。

```json
"formulas": {
  "50": {"a": 1728, "n": "2/3", "b": 5.02}   // r = 1728/(t^(2/3)+5.02)
}
```

## 注意事項

- 本ソフトは設計検討の補助を目的としています。**成果は必ず各県の最新基準・審査機関の指導と照合してください。**
- 円形オリフィスの未満管時は水没面積による近似、矩形は越流せき式（C=1.8）で扱っています。
- フォントは IPAゴシック（IPAフォントライセンス v1.0、`app/fonts/IPA_Font_License.txt`）を同梱しています。

## ディレクトリ構成

```
app/
  main.py            FastAPI（画面・計算API・PDF出力）
  engine/            計算エンジン（UI非依存・純粋関数）
    rainfall.py      降雨強度式・計画降雨波形
    runoff.py        洪水到達時間・流入ハイドログラフ（合理式連続モデル）
    infiltration.py  浸透施設（設計浸透量R・設計浸透強度Fc、有効降雨モデル）
    storage.py       H-A-V・オリフィス・貯留追跡（厳密法）
    simplified.py    簡便法
    spillway.py      洪水吐き・余裕高
    presets.py       県プリセット読込
    project.py       入力一式→全計算のオーケストレータ
  report/builder.py  PDF計算書組版（ReportLab）
  templates/ static/ 入力フォームUI
  fonts/             IPAゴシック（ライセンス同梱）
data/prefectures/    県プリセット（ibaraki.json / chiba.json）
tests/               pytest（基準書数値表との照合含む）
```
